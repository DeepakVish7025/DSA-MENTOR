🚀 DSA Mentor
DSA Mentor is a full-stack coding platform inspired by LeetCode, designed to help students practice Data Structures & Algorithms, participate in contests, learn through courses, and prepare for technical interviews—all in one place.

🔗 Live Demo: https://dsamazingleet.vercel.app/signup 🔗 GitHub Repository: https://github.com/DeepakVish7025/DSA-Mentor

📌 Features
🔐 Authentication System
User Registration & Login
Secure Logout functionality
Session handling and authentication
🏠 Homepage
Browse a wide range of DSA problems
Clean and user-friendly UI
Categorized problem listing
💻 Problem Solving
Dedicated problem page for each question
Code editor for solving problems
Run & submit solutions
📘 Editorial Section
Each problem includes:

✅ Video solution
✅ Reference solution by admin
✅ Explanation for better understanding
🤖 AI Chatbot
Helps users solve doubts
Assists in understanding concepts
Provides guidance during problem solving
📊 Submission History
Track all submissions with:

⏱ Time taken
💾 Memory usage
📈 Status (Accepted / Wrong Answer / etc.)
Helps users analyze performance

🏆 Contest System
Admin can create contests
Users can participate in real-time
Live leaderboard to track rankings
🎓 Course Section
Structured learning modules
Users can learn concepts step-by-step
🎖 Certificate generation after completion
🧠 AI Interview Feature
Practice interview-style questions
AI-assisted interview preparation
🛠 Tech Stack
Frontend:
